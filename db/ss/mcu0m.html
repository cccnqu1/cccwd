MCU0 微處理器 -- 超簡易的 16 位元處理器

指令格式

指令格式: OP C

OP 為四位元指令碼

C 為12位元常數，通常是記憶體位址。

每個指令均為16位元。

C 語言表頭宣告

檔案: mcu0.h

#include <stdio.h>

#define JMP  0x2
#define ST   0x3
#define CMP  0x4
#define JEQ  0x5
#define RET  0xE
