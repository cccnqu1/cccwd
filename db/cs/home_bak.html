關於本書

本書是金門大學陳鍾誠的「視窗程式設計、網路程式設計、遊戲程式設計」等課程的上課教材匯總，原稿採用 markdown+pandoc 的方式編排。

本書內容

  1. 簡介與開發環境
  2. 變數與運算式
  3. 流程控制
  4. 陣列
  5. 函數
  6. 物件
  7. 例外處理
  8. 檔案處理
  9. 資料結構
  10. 正規表達式
  11. 作業系統與 Thread
  12. 視窗程式簡介
  13. 時間驅動
  14. 設計文字編輯器
  15. DataGridView 與 ListView 元件
  16. 網路程式簡介
  17. UDP 程式設計
  18. TCP 程式設計
  19. HTTP 程式設計 (1) – Server
  20. HTTP 程式設計 (2) – Client
  21. XNA 遊戲引擎
  22. 結語
