邏漢語 Lohan (邏輯漢語)

邏漢語 Lohan 具有邏輯語的語法，但是卻可以直接寫入漢字的字詞。

邏萬語的文檔中可以加入註解，每一行在符號 // 之後的文字均為註解。

範例：

do citka lo plise .i // You eat apples.
do 吃    lo 蘋果  。
你 吃    位 蘋果  。

mi citka ci lo plise .i
mi citka 3  lo plise .i
我 吃    3  位 蘋果  。

填充規則

x1 comes/goes to destination x2 from origin x3 via route x4 using means/vehicle x5.

// x1 移動到目的地 x2, 從 x3 出發, 經過路徑 x4 , 使用方法/交通工具 x5.

一種辦法是用zo'e來充當sumti來填充那些被“省略”的位，zo'e的意思是“不必要的東西”。所以

la suzyn. klama la paris. la berlin. zo'e le karce
那 蘇珊   去    那 巴黎   那 柏林    啾   類 車

的含義就是：蘇珊坐車從柏林出發到巴黎（我們並不關心蘇珊途經何處）。事實上，zo'e是經常被隱含的，即便我們不明說出來。如果有人說klama，實際上他說的就是：

zo'e klama zo'e zo'e zo'e
啾   去    啾   啾   啾

大多數人都不希望一句話裡有太多的zo'e（當然，如果你願意這樣講也沒錯）。還有一種更流行的方法就是用位序號詞來標明位：fa、fe、fi、fo和fu。這些標記分別關聯到一個selbri的各個特定的位，無論它在句子中實際是什麼次序：fa代表第一個位，fe代表第二個位，以此類推。例如：

la suzyn. klama fu le karce
那 蘇珊   去    伍 類 車

蘇珊是坐車去的。

mo (做什麼, 問謂詞)

最後出場的是mo。它非常類似於ma，不同的是它代表selbri而不是sumti——在漢語裡就是“干什麼？”和“是什麼？”的區別（記住，“是什麼”和“干什麼”在邏輯語裡並沒有本質上的區別）。從邏輯的角度來說，我們可以將mo視為向某人問詢sumti之間的關系。

指稱詞

这里还有一些主要的冠词和类似冠词的单词：

la 被命名的 (那)
le 被描述的 (類)
lo 真实存在的 (位)
li 数字 (幾)

（lu不是冠词，它是一个引用标记）

la'e ……的代指 (那的)（la'e所指的是上一句话所指的，而di'u就是上一句里真实存在的单词）
le'e 定型的 (類的)
lo'e 典型的 (位的)

lai 被命名为……的集团 (那群)
lei 被描述的集团     (類群)
loi 真实存在集团     (位群)

la'i 被名为……组 (那組)
le'i 被描述的组 (類組)
lo'i 真实存在组 (位組)

lu'o，它可以把一个组转换成一个集团，而lu'a则把一个集团转换成一组独立的个体（“集合”和“解散”）。严格的来说，这些都不是冠词。

我们通过使用单词nu来解决这个问题，它的意思是——你可能猜到了——“事件（或状态）”。这被称为“谓词子句描述词”，常见的同类描述词还有ka（质量或属性）、ni（数量）等等（完整的清单请查阅The Complete Lojban Language第269页）。

如果你更喜欢使用12小时制，你需要通过一定的方法来区分AM和PM。在逻辑语里一种方便的方法就是使用cmene（我们随后会介绍一些关于cmene的补充知识）。所以“四点钟”就是la vocac.，“五点钟”就是la mucac.，以此类推（来自于gismu：cacra——小时）。对于11和12，我们需要额外的数字。幸运的是，逻辑语早有准备——数字进制可以扩展到16进制，所以我们还有一些“额外”的数字：

dau 10 fei 11 gai 12 jau 13 rei 14

// “上半天”和“下半天”我们就再加上lir.和lec.，意思是“早”和“晚”（来自gismu：clira和lerci）。所以la mucac.lir.就是“早上五点”

在逻辑语里，se是和te、ve、xe属于同一系列的单词，它们作用是将selbri的第一个位置分别和第二、第三、第四、第五个位置进行位置交换（这种类型的交换被称为“换位”）。

mi viska do // 我看到了你。
我 看到  你

do se     viska mi // 你被我看到了。
你 換2=讓 看    我

le nanmu cu klama lo barja // 那男人去了酒吧。
類 男人  素 去    位 酒吧

lo barja cu se  klama le nanmu // 酒吧就是那个男人去的（地方）。
位 酒吧  素 換2 去    類 男人

la spot.  mlatu la .abisinian. // “斯坡特”是一只阿比西尼亚品种的猫咪。
那 斯坡特 猫    那 阿比西尼亚品种
la .abisinian.     se  mlatu la spot. // 阿比西尼亚是猫咪“斯坡特”的品种。
那 阿比西尼亚品种  換2 猫    那 斯坡特

时间与事件，进阶二：项标签（sumti tcita）

逻辑语准备了一系列的结构词（cmavo），它们的作用是给selbri添加额外的位。

ti'u，意思是：“发生在……的时间”

mi klama ti'u la daucac. // 我到来的时间是10:00。
我 去    发生 那 10點鐘

klama在这里表达了六种事物间的关系：一个行者，一个目的地，一个出发地，一条路径，一种交通工具，还有一个“事件发生的时刻”。

日期

fe  lenu mi jbena [kei] cu detri fa  li pamu pi'e bi // 我出生（这个事件）的日期是八月十五日。
填2 事件 我 出生        素 日期  填1 粒 15   點的 8

但是使用se更容易些：

lenu mi jbena cu se  detri li pamu pi'e bi // 我出生（这个事件）发生的日期是八月十五日。
事件 我 出生  素 換2 日期  粒 15   點的 8

在上述两种情况里，都把lenu放在cu前面是很方便的——交换中有一个著名的逻辑语技巧：cu足以终止任何在它之前的语法结构，包括lenu mi jbena。

你也许猜到了：一定存在一个sumti tcita用来为"日期"服务。它就是de'i，它的工作方式与ti'u类似（请留意sumti tcita试图与selbri类似就如同他们所暗示的那样）。所以，另一种来告诉你我的生日的办法是：

mi jbena de'i li pamu pi'e bi
我 出生  日期 粒 15   點的 8
