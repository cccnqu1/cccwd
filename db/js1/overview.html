大綱

本課程可以當作「JavaScript 的入門磚」，採用 Node.js 作為主要開發平台，避免一下就將 HTML/CSS 等不相關技術納入，減少初學者的學習障礙。

我們會從基本的語法教起，並以 node.js 進行操作與實作，等到 JavaScript 中簡單的語法教完之後，隨即進入網頁前端的設計。

前端部分會從 Canvas 入門，接著才介紹 HTML/CSS 等標準技術，之後會採用 jQuery, Bootstrap 等框架進行搭配。

進入後端的伺服器設計部分時，採用 koa.js 作為主要框架，並搭配 mongoDB 作為資料庫，結合 socket.io 等通訊方式，著重在 Single Page Application 單頁應用上，以形成完整的系統。
