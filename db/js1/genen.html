習題：自動產生英文語句

提示：先用簡單的幾個字加上基本語法就行了，不用一下企圖心太大。

簡易語法

S = NP VP
NP = DET N
VP = V NP
N = dog | cat
V = chase | eat
DET = a | the

產生過程的範例

S = NP VP = (DET N) (V NP) 
  = (a dog) (chase DET N) 
  = a dog chase a cat
