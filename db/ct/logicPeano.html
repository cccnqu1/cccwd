從皮雅諾公理系統證明 1+1=2

在 《布林邏輯與推論系統 -- 何謂嚴格的數學證明？》 這篇文章中，我們已經說明了何謂邏輯推論，以及邏輯推論應用在數學證明上的方法。

現在、讓我們進一步的用這種邏輯推論的證明方式，想辦法證明一些非常簡單的數學定理。

最簡單的數學系統，大概就是包含加法與乘法的「算術系統」了，這個領域的數學通常稱為「數論」。

在「數論」當中，皮雅諾公設是一組經典的公理系統，包含下列邏輯法則。

皮雅諾公理系統 (Peano axioms)

「皮雅諾公理系統」的「一階邏輯」寫法如下，其中的 ∀ 是一階邏輯中的「對於所有」的意思，而 N 是指自然數，S(x) 代表 x 的下一個數字，也就是 x+1。

1. ∀x  ∈N. 0 ≠ S(x)
2. ∀x,y∈N. S(x) = S(y) ⇒ x = y
3. ∀x  ∈N. x + 0 = x
4. ∀x,y∈N. x + S(y) = S(x + y)

在「皮雅諾公理系統」裏，所謂的 1 其實是 S(0) 的簡寫， 2 則是 S(S(0)) 的簡寫，而 3 是 S(S(S(0))) 的簡寫，以此類推 ....

那麼、我們應該怎麼證明 1+1=2 這麼簡單的常識呢？

這件事情拿去問很多學過數學的人，都會把他們難倒的。但是現在我們就要利用「皮雅諾公理系統」證明「1+1=2」。

首先我們用比較快速的寫法進行一次證明。

簡短的證明：

1+1 = 1+S(0) = S(1+0) = S(S(0)+0) = S(S(0+0)) = S(S(0)) = 2

有些人可能看不出其中每個等號之間的關連，因此我們用更詳細的寫法將每個步驟之間根據的公理與定義寫下來，以下是詳細的證明過程。

詳細證明：

1+1 = 1+S(0)            ;  1 是 S(0) 的簡寫
    = S(1+0)            ;  根據公理 4 , 1+S(0) = S(1+0)
    = S(S(0)+0)         ;  1 是 S(0) 的簡寫 
    = S(0+S(0))         ;  交換律 : 嚴格來說交換律是需要證明的，如果無法證明就必須納入公理系統中
    = S(S(0+0))         ;  根據公理 4 , 0 + S(0) = S(0 + 0)
    = S(S(0))           ;  根據公理 3 , x+0=x  =>  0+0=0
    = 2                 ;  2 是 S(S(0))的簡寫。

透過上述的代換過程，您應開可以清楚的看出如何用「皮雅諾公理系統」證明出像 1+1=2 這麼簡單的定理了。

當初希爾伯特就是希望透過這樣嚴格的方式，讓整個證明過程可以轉化成一系列的變數取代與推論，這種方法就是所謂的公理化的數學建構方式了。

陳鍾誠 一個好的公理系統應該具有一致性，最好還能具有完備性。

一致性是指推出來的定理之間不會有矛盾。

而完備性則是指所有「真的事情」(定理) 都可以被推出來。

德國數理邏輯大師戈特洛布·弗雷格（Frege）曾研究用集合論去描述數理邏輯，為此他還寫了一本書。他在給羅素的信中提到他的工作時說他為此構造了一個特殊的集合（A），這個集合由所有不包含自己的集合構成。也就是說，集合A的元素X是一個集合，X自己不是自己的元素，即X \not\in X。羅素在回信中講述了前面的理髮師的故事。聰明的弗雷格看出了這實際上是指出了他所構造的集合A的問題：如果A \not\in A，那麼根據定義A應該包含A，即A \in A；但是如果A \in A，那麼同樣根據定義A又不應該包含A，即A \not\in A。可此時弗雷格的書已經付印，修改已經是不可能的了，弗雷格只能在書中加一個後記並寫到：在工作結束之後而發現那大廈的基礎已經動搖，對於一個科學工作者來說，沒有比這更為不幸的了。
