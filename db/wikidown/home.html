/ 關於 wikidown

關於 wikidown

  • wikidown 是一個使用 markdown 語法編輯的 維基 （ wiki） 系統，採用 node.js 作為伺服器 ( wikiServer.js )。
  • wikidown 在 github pages 上的首頁為 http://ccckmit.github.io/wikidown1/public/wikidown.html
  • wikidown 的原始碼位於 https://github.com/ccckmit/wikidown1
  • wikidown 的專案的程式設計原理請參考文章 -- 使用 node.js 設計 wikidown 網誌系統
  • wikidown 顯示連結時， 淡咖啡色 代表內部連結，淡藍色 代表外部連結。
  • wikidown 支援內部連結語法，使用時請用 `![[圖片名稱]](filename)` 符號將數學式框起來，圖片請放在 web/img/ 資料夾底下，請參考 內部圖片範例 。
  • wikidown 支援latex數學式，使用時請用 `$... {%=title%} {%=pathLink%} 符號將數學式框起來，wikidown會用MathJax將數學式呈現出來，請參考 數學式範例 。
  • wikidown 支援程式碼自動上色，採用 highlight.js 對程式碼進行上色動作，請參考 程式碼範例 。
  • wikidown 的作者為 陳鍾誠 。
