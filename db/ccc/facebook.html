管理學上我認為最重要的一條法則：彼得定律
-------------------------------------------------------
當寫程式的好手被升為經理之後，他反而不能去寫程式了，結果他成為一位不好的管理者，直到被公司開除或者該公司倒閉為止。
-------------------------------------------------------
彼得定律
管理學家勞倫斯·彼得從大量失敗案例中總結出一條原理：在一個等級制度中，每個雇員都傾向於晉升到不能勝任的地位之後卡住，再也升不上去了。
而整個組織也因此僵化而衰敗，因為絕大部份的職位都由不適任的人所擔任。
-------------------------------------------------------

我現在和學生聊天時常常會灌輸他們一些異端邪說：
例如有位同學打算轉台清交，因為他認為念比較好的學校後，出國留學被好學校接受的機會大增。
他打算念完大學後去日本或德國留學。
這將會是四年後的事情！
但是我發現他對德國的大學和體制都沒什麼概念。
然後我就給他一個瘋狂的提議：為什麼不現在就出發呢？
先休學後直接去德國。
或者先利用寒暑假去德國念語言學校，順便進大學修課。
這樣就不需要等四年後，而且也會比較清楚德國教育體制的情況。
如果我現在和他年齡一樣，我應該會做這種事，只要經濟問題不太嚴重的話。
這算是某種異端邪說嗎？

臉書的用法很多，大致分兩種：
第一種： 理想型，在臉書上主張自己的理想，不考慮會得罪別人，甚麼都講的人，這種人通常甚麼人加他好友都會加。
第二種： 怕得罪人型，po 文時想東想西，深怕得罪朋友，加好友也只加認識的人。
第一種有點像男生，兩個男生打架打一打之後，居然變成了好朋友，這是很多女生無法理解的事情。
理想型的兩個人在臉書吵了半天之後，隔天又繼續聊天，好像沒有發生過甚麼事一樣。
第二種有點像女生，兩個閨密女友 A 和 B，雙方都深怕得罪對方，但是有一天 A 去日本沒有幫 B 買東西，結果兩個閨密竟然積怨在心，最後反目成仇。
怕得罪人型的人，最後在臉書上，通常只有吃喝玩樂的照片分享，以及一大堆稱讚與羨慕的訊息。(因為這樣絕對不會得罪人喔！)

這幾天狂招人忌，讓我想起「不招人忌是庸才」這句話！
那「招人忌」可以用來推論「不是庸才」嗎？
非也非也！
數理邏輯說， p => q 可推論 -q => -p
所以「不招人忌 => 是庸才」等同於「不是庸才 => 招人忌」。
但這個和「招人忌 => 不是庸才」 是不相等的。
因為、「招人忌 => 是庸才」也是有可能發生的。
