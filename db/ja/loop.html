迴圈

範例：從 1 印到 9 (使用 for)

class For1 
{ 
  public static void main(String[] args) 
  {
    for (int i=1; i<=9; i++)
      System.out.println("第"+i+ "行");    
  }
}

範例：從 1 印到 9 (使用 while)

class While1 
{ 
  public static void main(String[] args) 
  {
    int i=1;
    while (i<=9) 
    {
      System.out.println("第"+i+ "行");    
      i++;
    }
  }
}

範例：九九乘法表

public class Loop99
{
  public static void main(String[] args)
  {
    for (int i=2; i<=9; i++)
      for (int j=1; j<=9; j++)
        System.out.println(i+"*"+j+"="+i*j);
  }
}
