前言

一般的軟體工程書籍，都是從理論開始探討的。而很多軟體工程的書籍，幾乎都在講原理原則，甚至完全沒有實務操作。

為了平衡這種過於理論的觀點，我決定寫下這本書。

在這本書裡，所有的主題都有實務操作。

要有實務操作，就不能避開程式語言，於是我們選定了 javascript 作為本書的操作核心，並且搭配 node.js, io.js 這樣的伺服端平台，以進行實務演練。

我們希望，透過這種方式，能讓軟體工程不再是一門理論課，而是可以實際派上用場的知識。

當您用過了這些實務工具，再回頭去看理論性的議題，相信必然會有不同的感受才對！
